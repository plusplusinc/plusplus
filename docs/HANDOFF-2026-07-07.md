## Operational knowledge that isn't in CLAUDE.md

- **Workflow rhythm**: per-PR `claude/*` branches → dispatch ci.yml (workflow_dispatch runs ui-test too; plain pushes skip it) → self-merge (squash) on green → batch TestFlight dispatches; build number = run number. Chain `send_later` timers (~25 min) to follow CI/uploads across turns.

- **Fresh installs**: builds ≥15 reset stores (entity renames); the store-recovery fix in ≥16 makes future update-in-place safe.
- **Docs-only pushes skip CI** (`paths-ignore: **.md, docs/**`) — LICENSE files are NOT .md, so those runs do execute.
